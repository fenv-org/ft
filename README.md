# ft

## How to install

1. Runs the installer
   ```bash
   curl -fsSL https://ft.install.jerry.company | bash
   ```
2. Add the flutter-test to your path
   ```bash
   echo 'export PATH="$HOME/.flutter-test/bin:$PATH"' >> ~/.bashrc
   source ~/.bashrc
   ```

## How to use

1. Run the `ft`
   ```bash
   ft
   ```
2. For more options
   ```bash
   ft --help
   ```
